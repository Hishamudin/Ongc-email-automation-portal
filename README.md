# Ongc-email-automation-portal
A centralized portal for managing employee and group email ID requests with admin approval workflow, developed during ONGC internship july 2024


A web-based system developed during my internship at ONGC to streamline internal email ID requests.
## Features
- Employee Mail ID Creation
- Mail ID Modification
- Group Mail ID Requests
- Special Mail ID Requests
- Admin Approval Dashboard
## Tech Stack
- React + Vite
- Tailwind CSS
- TypeScript
- PHP & MySQL (backend concept)
